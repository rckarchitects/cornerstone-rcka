Welcome to Cornerstone

Cornerstone is RCKa's bespoke architectural practice management system It is a web-based management system for small architectural practices. It has been developed by RCKa architects over several years to assist with day-to-day tasks that are often laborious and time-consuming. The Architects' Journal has recently published an article about the software and our reasons for making it open-source. You can read this article here: http://bit.ly/1LvQgS5 Please note that the platform is still at a very early stage of development, and so you are likely to find numerous bugs, glitches and other issues that may mean it doesn't quite function how you expect. You might even struggle to install it. There's also no documentation - at least for the time being. But we're working on all of these things and hope to make it easier to use and install soon. A list of some of the features included within Cornerstone:

  • Time tracking / analysis
  • Invoicing
  • CRM (Customer Relationship Management) / Contacts database
  • Drawing / document issue
  • QMS (Quality Management System)
  • Task management
  • Tender preparation
  • Project journals
  • Staff holiday management
  • Expenses management
  • Planning condition tracker
  • Project checklist

Cornerstone runs on a standard web server using MySQL, Apache and PHP, so some knowledge of these things will be required to get it up and running. Probably best to come back soon when we've made more progress in simplifying the installation process.
